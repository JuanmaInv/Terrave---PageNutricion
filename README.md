# NutriLen

Plataforma web para evaluación sensorial de medallones de lenteja.
Integrador entre Nutricion e Ingenieria en Sistemas De Informacion.

## Stack

### Frontend
Next.js + React + TailwindCSS

### Backend
NestJS

### Base de datos
Supabase PostgreSQL

### Autenticación
Clerk

### Testing
Playwright + Unit Testing

### Deploy
Frontend: Vercel  
Backend: Render
